## Key Features

The new Instruqt lab format provides:

- **Resource-based Configuration**: Define labs, sandboxes, tasks, and content as discrete HCL resources
- **Structured Content Organization**: Organize instructions, scripts, and assets in logical directory hierarchies  
- **Advanced Interactivity**: Create tasks with validation scripts, quizzes, and interactive UI components
- **Flexible Layouts**: Configure multi-panel interfaces with terminals, editors, browsers, and custom tabs
- **Migration Support**: Automated tools to convert legacy YAML tracks to the new HCL format
