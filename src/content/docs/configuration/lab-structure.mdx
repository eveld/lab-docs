This page covers the structure and organization of Instruqt labs using the HCL format.

## Core Configuration Files

Every lab consists of several core HCL configuration files:

- **`lab.hcl`** - Main lab resource definition
- **`sandbox.hcl`** - Infrastructure and sandbox resources  
- `chapters.hcl` - Content organization and structure
- `tasks.hcl` - Interactive activities and validation
- `tabs.hcl` - User interface configuration

  - files/ Configuration files and templates

## Resource Relationships

Lab resources are interconnected through references:

1. **Lab** references chapters and defines overall structure
2. **Chapters** contain pages and define content flow
3. **Pages** reference tasks and instruction content
4. **Tasks** reference validation scripts and activities
5. **Sandbox** resources provide the runtime environment

This structure provides clear separation of concerns while maintaining flexibility for complex lab scenarios.
